# Project 4 — Dynamic Web Application using EC2 and RDS

## 🚀 Overview
This project demonstrates a full-stack cloud application where a web server communicates with a managed database.

## 🧱 Architecture
User → EC2 (PHP Web Server) → RDS (MySQL Database)

## 🛠️ Technologies Used
- AWS EC2
- AWS RDS (MySQL)
- PHP
- MySQL

## ⚙️ Implementation Steps
1. Launched EC2 instance and installed Apache, PHP
2. Created RDS MySQL database
3. Configured security groups for connectivity
4. Connected EC2 to RDS
5. Developed PHP application
6. Implemented CRUD operations

## ✨ Features
- Add, update, delete users
- Persistent database storage
- Dynamic content rendering

## 📌 Learning Outcomes
- Database integration with cloud apps
- Backend development basics
- Managed database services
